# Open questions

For the planning/grilling session after the autonomous build.

1. **PWC add-on test mode.** The real assessment adds ~20 PWC questions
   for a PWCL. Should the mock test grow a "boat + PWC" variant (50+20)
   and a PWC-only 40-question variant, mirroring the real structures?
   The bank tags PWC questions, so this is mostly UI.

2. **Practical assessment content.** The MSQ practical checklists (RMDL
   and PWC) are documented in the research. Worth a read-only "what the
   on-water day looks like" page, or out of scope for a quiz app?

3. **Water limits geography.** The water limits maps PDF is graphical;
   the knowledge base only carries the definitions and a list of mapped
   port areas. Are location-specific questions (e.g. Moreton Bay
   boundaries) worth hand-authoring against the real maps?

4. **Question depth vs the 100 percent bar.** The real CAT retrains on
   any miss. Should mastery in this app require more than one correct
   answer (e.g. twice in a row) given the target is perfection?

5. **Fact drift.** Licence fees index every July; rules change. What is
   the maintenance story - a periodic re-verification pass against
   legislation.qld.gov.au? The rule-corrections doc records what was
   verified and when (July 2026).

6. **Sharing with learners.** Is this for Sam only, or worth sharing
   (e.g. with the boating community)? Public Pages site + flag links
   already support outside users; a feedback loop needs the GitHub issue
   tracker to stay open.

7. **Local marine zones.** Gold Coast and other local rules (PWC
   curfews, Lake Wivenhoe zone) are only lightly covered. A "local
   rules" topic could matter for real-world safety even though the CAT
   tests statewide rules.
